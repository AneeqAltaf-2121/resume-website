# Personal Portfolio Website

A modern personal portfolio built with **Python** and **Streamlit** that showcases my background, software engineering experience, AI/ML projects, technical skills, certifications, and leadership activities.

## 🌐 Live Demo

**Website:** https://aneeq-altaf-resume-website.streamlit.app/

## 🚀 Features

* Professional portfolio landing page
* About Me section
* Software Engineering internship experience
* AI & Machine Learning project showcase
* Technical skills and certifications
* Leadership and Honors section
* Responsive Streamlit interface with custom styling
* Profile images and project visuals

## 🛠️ Tech Stack

* Python
* Streamlit
* HTML/CSS (embedded in Streamlit)
* Base64 image encoding

## 💻 Running Locally

Clone the repository:

```bash
git clone https://github.com/AneeqAltaf-2121/<repository-name>.git
cd <repository-name>
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

The application will open in your browser at:

```
http://localhost:8501
```
